# Mikhak
Mikhak is a simple monoline semi handwriting Arabic-Latin typeface in <a href="https://aminabedi68.github.io/Mikhak/">5 weights</a> and <a href="https://aminabedi68.github.io/Mikhak/VF.html">variable</a> version.
